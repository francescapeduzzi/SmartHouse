# SmartHouse

Progetto per il corso di sistemi operativi. 
Lo scopo del programma è quello di simulare una SmartHouse in cui è possibile, dopo la configurazione iniziale, controllare l’accensione dei led, visualizzarne lo stato corrente, monitorare la temperatura e modificare o visualizzare la configurazione sulla eprom. Inoltre la pressione del pulsante consente di attivare il buzzer e in contemporanea di accendere e spegnere tutti i led. Nel caso in cui vengano accesi tutti i led verrà attivato il timer per spegnere le luci.
Tutti i comandi vengono inviati tramite terminale. Digitando il comando esc si esce dal programma.
